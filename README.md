HR Analytics & Employee Performance Tracker (2026)									
									
📋 Project Overview									
									
This project is a comprehensive HR Management Tool developed using Microsoft Excel 2007. It is 									
designed to help HR professionals and managers track employee performance, analyze salary 									
distributions, and identify potential attrition risks within an organization.									
									
The analysis is based on a dataset of 50 employees across two key regions: Indore and Udaipur.									
									
									
									
🗝️ Key Features & Insights									
									
Interactive Dashboard: A clean, grid-free visual interface featuring key performance indicators (KPIs).									
									
Attrition Risk Analysis: A logical system that identifies "High Risk" employees (12%) based on low .									
		        performance and high leave frequency							
									
Regional Workforce Distribution: Balanced insights showing a 52% (Indore) and 48% (Udaipur) 									
			        employee split.						
									
Department-wise Payroll Tracking: Deep dive into salary costs, identifying IT as the highest 									
			           expenditure department.						
									
Performance Metrics: Real-time tracking of the average employee rating, currently at 3.86/5.0.									
									
									
🧪 Technical Implementation (The "Backend" Logic)									
									
Data Structuring: Managed raw employee data including IDs, Departments, Joining Dates, and 									
	                Performance Scores.								
Advanced Formulas:									
									
Attrition Logic: Used IF(AND()) functions to flag critical staff.									
									
Salary Bracketing: Categorized employees into High, Mid, and Low salary groups.									
									
Pivot Tables: Automated data aggregation for regional and departmental summaries.									
									
Data Visualization: Utilized Clustered Bar Charts, Pie Charts, and Doughnut Charts optimized for 									
		Excel 2007.							
									
🛠️ Tools Used									
									
Microsoft Excel 2007 (Core Analysis & Visualization).									
									
Pivot Table Engine (Data Summarization).									
									
Logical Functions (Predictive Analysis).
